# QIQA Demo Online Forms

Static HTML demo forms for testing the QIQA Chrome form-filling extension.

These pages contain no patient data and do not submit data to a server.

## Pages

- `index.html`
- `login.html`
- `target-form.html`
- `medical-intake-online-form.html`

The form pages use a demo-only sessionStorage login gate. Any email/password works on `login.html`; no credentials are sent to a server.

## GitHub Pages

Enable Pages from repository settings:

- Source: deploy from a branch
- Branch: `main`
- Folder: `/root`
